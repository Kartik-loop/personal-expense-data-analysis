
# Personal Expense Analysis Project

## Description
This project demonstrates a complete data analysis pipeline using Python and Google Colab.
The analysis includes data loading, cleaning, exploration, visualization, and insight generation.

## Dataset
- Source: Kaggle
- Type: Personal expense and income dataset
- Format: CSV

## Project Structure
data/ - Dataset
visualizations/ - Generated charts
report/ - Analysis report
analysis.ipynb - Main notebook
requirements.txt - Dependencies

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Google Colab

## How to Run
1. Upload the dataset using the Colab upload function.
2. Run all cells in `analysis.ipynb`.
3. View generated insights and visualizations.

## Visualizations
- Category-wise expense distribution
- Monthly expense trend over time
  
## Dataset
The dataset file `expenses_dataset.csv` is included in the repository root and is loaded directly in the analysis notebook.

## Results
- Category-wise expense analysis
- Monthly expense trend visualization
- Income vs savings relationship
