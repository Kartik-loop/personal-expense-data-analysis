# Personal Expense Data Analysis Report

## Project Overview
This project performs a complete data analysis on a personal expense dataset obtained from Kaggle. 
The goal is to understand spending behavior, savings patterns, and monthly expense trends.

## Data Description
The dataset contains information about income, expenses, savings, and multiple expense categories such as food, rent, shopping, and transport.

## Data Cleaning & Validation
- Missing values were handled using median (numerical) and mode (categorical) methods.
- Date columns were converted to proper datetime format.
- Negative savings values were corrected.
- Duplicate records were removed.

## Key Insights
- Rent and food contribute the highest share of total expenses.
- Higher income does not always result in higher savings.
- Monthly expenses show noticeable fluctuations, indicating seasonal or lifestyle-based spending patterns.

## Conclusion
This analysis highlights the importance of tracking expenses to improve financial planning and savings habits. 
The insights can help individuals make informed financial decisions.
